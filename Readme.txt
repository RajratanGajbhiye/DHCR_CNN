Devanagari character Recognition:

- The dataset can be downloaded from the following link: https://archive.ics.uci.edu/ml/datasets/Devanagari+Handwritten+Character+Dataset

Required Libraries:
- Python 3.x
- TensorFlow
- Keras
- OpenCV
- Matplotlib

Dependencies:

- Make sure to install the required dependencies using below command:

pip install tensorflow matplotlib opencv-python
